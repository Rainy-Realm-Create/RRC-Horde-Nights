# Welcome to Horde Nights!

This datapack/mod adds the ability to encounter a Blood Moon. The Blood Moon happens once every 30-45 days based on difficulty.

During Blood Moon events, you will have to face The Horde, and members of The Horde will be enhanced based on difficulty.

If you wish to summon a Blood Moon yourself, instead of waiting, you can perform the Blood Ritual by dropping a diamond on a soul campfire.
Make sure it's enabled in the config!

We have also added 2+ advancements on our own, plus 1+ Super Challenge(s) when paired with Blaze & Cave's Advancement Pack(listed below).

## Config

By running the command ```/function h-nigh:config``` you will open the customization menu. This is a datapck based project, so text file configs are impossible.

You can customize the pack difficulty, or leave it dependant on your game difficulty.

You can customize the interval of Blood Moons, and toggle features such as Cowardice & The Blood Ritual.

## Integrations

Horde Nights has integrations with several other packs. Please bear in mind sometimes this list may be a little out of date.

- [Mo-Variants](https://modrinth.com/datapack/mo-variants) Integration for 1.20+
  - This integration reduces issues with some variants, and even adds a couple unique Horde Effects for some other variants.
  - Works best with Mo-Variants 7.7.0 or Higher(this version is currently unreleased)
- [BACAP](https://modrinth.com/datapack/blazeandcaves-advancements-pack) Integration for 1.20+
  - This integration adds 1+ Super Challenge(s) to complete.
  - Not using this integration creates a cosmetic error in log, ignore it.
- [Creeper Overhaul](https://modrinth.com/mod/creeper-overhaul) Integration for 1.20/1.20.1 & 1.21/1.21.1
  - This integration recruits 13/14 new creepers to The horde.
  - Not using this integration creates a cosmetic error in log, ignore it.

Only these projects are guarenteed to work with this pack. While we do our best to maintain decent compatability standards, things may interact weirdly. Please report any incomptabilities or suggest any ideas for official integration improvements in our discord!
